HOW TO USE THIS PACKAGE

To handle ACE archives from PeaZip, you will need to copy the "unace" directory
into PeaZip's application path (peazip)/res/bin/ folder.

For PeaZip installable versions the plugin folder usually is located in
/usr/lib/peazip/res/bin/ (Linux)
/Applications/peazip.app/Content/MacOS/bin/ (macOS)
C:\Program Files\PeaZip\res\bin\ (Windows)

HINT: From PeaZip main menu > Help > About and from Settings > About, it is possible to 
directly open the plugin folder and to check if unace plugin is installed.
